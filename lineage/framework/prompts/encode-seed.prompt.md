Append a new entry to the **Evolution Log (section 8)** of [seed.md](../../seed.md) recording what
was done this conversation. Do **not** modify any other section — sections 1–7 are the DNA blueprint
regenerated by the `sync-seed` skill.

## Instructions

### 1. Gather what happened this session
Collect from the conversation:
- **Tools run**: each tool called, key inputs, and outcome
- **Files created or modified**: paths + one-line description
- **Decisions made**: any branching choices
- **Session label**: use the argument if provided; otherwise derive a short label from the primary action

### 2. Format the entry

```
---

## <Session Label> — <YYYY-MM-DD>

<1–2 sentence summary.>

<For each significant tool call:>
Ran <Tool Name>
Completed with input: { <key inputs as JSON> }
<outcome in plain English>

<For each file created/modified:>
- Created/Updated `<path>`: <one-line description>
```

### 3. Append to the Evolution Log
- Read [seed.md](../../seed.md).
- Append the new entry at the **bottom of section 8 (Evolution Log)**, after the last existing entry.
- Do **not** modify sections 1–7 or any earlier log entry.

### 4. Confirm
Report what was appended: the session label, date, number of tool calls recorded, and files listed.
